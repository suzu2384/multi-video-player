複数動画同期プレーヤー

使い方
1. index.html をブラウザで開きます。
2. 動画を複数選択するか、画面へドラッグ＆ドロップします。
3. 通常は「複数動画モード」で、読み込んだ動画を一覧・同期再生できます。
4. 並べたい動画2本の「並列に選択」をオンにし、「並列動画モード」を押します。
5. 並列表示サイズで、左右を合わせた映像領域の横幅・縦幅を指定できます。
6. 各動画の「並列表示の切り抜き」で、拡大率・横位置・縦位置を調整できます。

補足
- 動画は外部へアップロードされず、ブラウザ内で処理されます。
- 指定サイズがブラウザ画面より大きい場合、縦横比を保ったまま表示だけ縮小されます。
- ブラウザが対応していない動画形式やコーデックは再生できません。

更新内容:
- ループ再生操作を削除
- 動画本数に応じて1〜6列を自動設定
- 列数を手動変更した後は設定を維持
- 操作バーに動画本数を常時表示
- モード復帰時の説明メッセージを削除


書き出し速度について
現在の書き出しはCanvas + MediaRecorderによる実時間録画です。ブラウザだけで完結し、音声ミックスや表示位置を確実に反映できますが、動画時間より短時間での書き出しはできません。高速書き出しにはFFmpeg等のネイティブエンコーダーを利用する別構成が必要です。
