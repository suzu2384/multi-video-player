複数動画同期プレーヤー（フレーム同期型リアルタイム合成・試験版）

使い方:
1. index.html を開く
2. 動画を追加する
3. 並列にする2本を選ぶ
4. 並列モードへ切り替えて再生する

並列モードでは、2本の video 要素を直接表示せず、各動画の表示済みフレーム時刻を監視し、
時刻差が許容範囲に入ったフレームだけを Canvas に左右合成して表示します。
片方のフレームが遅れた場合は、先に進んだ映像だけを表示せず待つため、
従来より開始時の見た目のずれを抑えます。

注意:
- iPhone Safariでの試験的な方式です。端末性能、動画形式、解像度によってはコマ落ちします。
- 2本のデコード自体は同時に行うため、完全な1ストリーム再生より負荷が高くなります。
- 音声は最初の同期フレームが描画されるまでミュートし、同期後に両方を再開します。
- GitHub Pagesへは index.html / styles.css / app.js をまとめて上書きしてください。
